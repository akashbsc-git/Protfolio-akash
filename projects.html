PROJECTS

1️⃣ Personal Portfolio Website

Designed and developed a fully responsive personal portfolio website using HTML5, CSS3, JavaScript, and Bootstrap. The website is structured with modern layout systems including Flexbox and CSS Grid to ensure clean alignment and consistent UI design across multiple screen sizes.

Integrated interactive components using JavaScript to enhance user experience. The website is optimized for mobile, tablet, and desktop devices following responsive design principles. Version control and project management were handled using GitHub for source code maintenance and updates.

The portfolio serves as a professional profile platform showcasing personal details, academic background, technical skills, projects, internship experience, and certifications in a structured and user-friendly format.

2️⃣ Hotel Room Booking System (Full Stack Academic Project)

Developed a complete hotel management web application using PHP, HTML, CSS, JavaScript, Bootstrap, and MySQL database. This system was designed to manage hotel room operations efficiently.

Key functionalities implemented:

  • Room check-in and check-out management
  • Customer billing system with receipt generation
  • Automated bill calculation based on stay duration
  • Print-ready receipt for check-in and check-out
  • Hotel report generation for administrative monitoring
  • User authentication and role-based access control
  • Add, edit, and remove room details
  • User management (add/remove users)
  • Password change and forgot password system
  • Settings module for system configuration

The application follows structured backend processing using PHP and MySQL for data storage, retrieval, and management. Authentication layers were implemented to ensure secure access control between users and administrative roles.

The system is fully responsive and optimized for multiple devices, maintaining consistent layout and performance across desktops, tablets, and mobile screens.
